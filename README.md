# quiz-app-kbc

deployed using netlify:
https://kanishk-quiz.netlify.app/

this app is running on react-vite

to run:

cd frontend
npm i
npm run dev

on browser go to:
localhost:5173

app will run.

app usage:

1. scan the code to enter quiz from phone
2. choose options hit submit 
3. play again if you want to.
